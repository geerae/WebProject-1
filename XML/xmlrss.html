XML Syntax Rules

6 important rules that must be adhered to in order to create a valid XML file.

Rule 1: All XML elements must have a closing tag.

In XML, it is illegal to omit the closing tag.

Rule 2: XML tags are case sensitive.

Opening and closing tags must be written with the same case.

Rules 3: All XML elements must be properly nested.

If the first element is started first, it should be the one at the end.

Rule 4: All XML documents must have a root element.

Must contain one root element(parent of all other elements).

Rule 5: Attribute values must always be quoted.

In XML, the atribute values must always be quoted.

Rule 6: The XML Prolog (optional)

This is optional, but if it exists it must come first in the document.
